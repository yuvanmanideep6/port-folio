Projects

FotoFam

FotoFam is a collaborative photo-sharing web app built during our second year of college. Designed with simplicity in mind, it allows multiple family members to post and view photos under a single shared account — no need for individual logins. Think of it as a mini family Instagram, where moments can be uploaded, viewed like posts, and cherished together. It’s a compact full-stack project aimed at building closer digital bonds within families.

Brownstone Boys

Brownstone Boys is a fully functional e-commerce web application developed as a student project to demonstrate core full-stack development skills. The platform allows users to browse, add to cart, and purchase aesthetic clothing items such as shirts, pants, and footwear. Built with a focus on responsive design and intuitive UX, the application simulates a real-world online store, complete with user authentication, product filtering, shopping cart management, and order checkout flow.

GitHub
To-Do List

To-Do List is a simple yet functional web application built to practice front-end development concepts such as DOM manipulation, state handling, and dynamic rendering. This project allows users to create, edit, and delete daily tasks in a clean, distraction-free interface. Developed as part of a learning sprint, the app focuses on core JavaScript logic, responsive design, and minimal but intuitive UI to simulate the core features of a productivity tool.
